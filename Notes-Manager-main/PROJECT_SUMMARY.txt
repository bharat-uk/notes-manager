   Schema:
   CREATE TABLE notes (
       id INTEGER PRIMARY KEY AUTOINCREMENT,
       title TEXT NOT NULL,
       content TEXT NOT NULL,
       created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
       updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
   );
   
   Operations:
   ├─ CREATE: addNote() inserts new note

   └─ DELETE: deleteNote() removes note

                        HOW TO RUN IN 3 STEPS
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Download JDBC Driver (once)
────────────────────────────────────
Open PowerShell in project folder and run:

mkdir lib; wget https://github.com/xerial/sqlite-jdbc/releases/download/3.44.0.0/sqlite-jdbc-3.44.0.0.jar -O lib/sqlite-jdbc-3.44.0.0.jar


STEP 2: Compile & Run
─────────────────────
Double-click: build_and_run.bat

OR in PowerShell:
powershell -ExecutionPolicy Bypass -File build_and_run.ps1


STEP 3: Enjoy!
──────────────
Application window opens immediately ✅
